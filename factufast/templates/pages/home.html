Features

Factufast is a free and open source invoicing software. It is designed to be easy to use and to be self-hosted.

Easy to use

Factufast is designed to be easy to use. It is a web application that can be used from any device.

Self-hosted

Factufast is a self-hosted application. You can install it on your own server and you can access it from anywhere.

Open source

Factufast is an open source project. You can contribute to the project or install it on your own server.
